# Envio de Mensagens

Pequena aplicação web para enviar e gerenciar mensagens em uma lista. Desenvolvida com HTML, CSS e JavaScript puro.

## 📋 Funcionalidades

- ✅ Adicionar mensagens à lista
- ✅ Editar mensagens existentes
- ✅ Excluir mensagens
- ✅ Interface responsiva e acessível
- ✅ Armazenamento local no navegador

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e responsividade
- **JavaScript** - Funcionalidades interativas

## 📦 Estrutura do Projeto

Av1-DW/
├── Index.html # Página principal
├── css/

└── README.md # Este arquivo


## 🚀 Como Usar

1. Abra o arquivo `Index.html` no navegador
2. Digite uma mensagem no input
3. Clique em "Adicionar" para inserir na lista
4. Use os botões para editar ou excluir itens

## 📝 Requisitos

- Navegador moderno com suporte a ES6+
- JavaScript ativado
